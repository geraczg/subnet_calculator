Cómo funciona la calculadora

  1. Ingresa la dirección IP base.
  2. Elige el prefijo CIDR de la red.
  3. Selecciona FLSM o VLSM según tu caso.
  4. Indica cuántas subredes o hosts necesitas.
  5. Presiona calcular y revisa el resumen y la tabla.

Conceptos clave

  • Red: Dirección que identifica el inicio de una subred.
  • Máscara: Valor que separa la parte de red y la parte de hosts.
  • Prefijo: Número de bits usados para identificar la red.
  • Broadcast: Dirección final usada para enviar a todos los hosts del bloque.
  • Hosts útiles: Direcciones disponibles para asignar a dispositivos.
  • Wildcard: Máscara inversa que indica qué bits pueden variar.
